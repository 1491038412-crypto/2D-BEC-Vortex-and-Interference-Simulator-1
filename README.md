# Cold Atom Lab ⚛️

An interactive simulator for Bose-Einstein condensate (BEC) dynamics, built with Streamlit.

## Features
- **Interference**: Two offset wave packets produce matter-wave interference fringes.
- **Vortex**: A rotating obstacle stirs quantized vortices in a harmonic trap.
- **Free Expansion**: A condensate expands freely in space.

## Physics
Solves the 2D Gross-Pitaevskii equation via the split-step Fourier method.

pip install -r requirements.txt
